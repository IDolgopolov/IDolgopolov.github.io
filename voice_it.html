VoiceIt

Статус. Завершен 10.09.2020 - 30.12.2020

Социальная сеть

VoiceIt - социальная сеть, которая позволяет обмениваться пользователям голосовыми историями.

Истории - популярный формат с короткими смешными или познавательными видеороликами для друзей или блогеров. Голосовые истории - ответвление от основной идеологии историй, исключающее видео-составляющую и делающее упор на аудио-формат.

Голосовые истории - короткие аудио-записки, в которых содержаться истории друзей, каверы на любимые песни и любой другой контент от блогеров.

Это часть обучающего проекта

Мобильное приложение VoiceIt разрабатывалось командой из четырех человек. Оно было контрольным проектом для курса по Android-разработке от компании Mail.ru и московского вуза МГТУ им. Баумана на проекте "Технопарк".

Архитектура

Приложение построено на принципах архитектуры MVVM и паттерна Single Activity.

Разработка велась командой, поэтому было принято решение делить весь код на модули. Это ускорило процесс разработки, позволило в малые сроки получить чистую архитектуру и стабильное приложение.

Модули:

  • FirebaseLoader - работа с удаленной БД, с моделями и файлами
  • Player - воспроизведение аудио
  • RoomCashe - кэширование аудио
  • Main - модели, верстка, каркас

Библиотеки и технологии

  • Glide - загрузка фото
  • Firebase Auth - авторизация пользователя
  • Firebase Storage - хранение аудио и фото
  • Firebase Firestore - хранение моделей
  • Lifecycler - для поддержки MVVM
  • AvatarImageGenerator - для генерации дефолтных картинок
  • Exoplayer - для загрузки и воспроизведения аудио
  • Firebase Analytics - для аналитики
  • Room - для кэширования
  • MaterialCompat - дизайн
  • GitHub - ведение проекта с множеством веток
  • Многопоточность и сервисы
  • Notification API - для отображения прогресса через уведомления
